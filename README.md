## Userscripts:
* **netflix-title**<br/>
  Adds show titles on netflix.
* **zoro.to**<br/>
Fixes titles on zoro.to.<br/>
Format: '{ShowTitle} [10/12]'.
* **youtube-title**<br/>
  Removes ' - YouTube' from youtube titles.
